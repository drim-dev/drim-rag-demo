# Практические приложения RAG — План реализации

**Проектный документ:** `docs/designs/2026-02-24-rag-practice-apps.md`
**Оптимизирован для:** Параллельного выполнения агентами (3 агента, по одному на приложение + общая настройка)

---

## Карта зависимостей

```
Фаза 0 (Общая настройка)
    │
    ├──→ Фаза 1A (Прил. 1: Ядро)
    │        │
    │        └──→ Фаза 1B (Прил. 1: Интерфейс + Финализация)
    │
    ├──→ Фаза 2A (Прил. 2: Ядро)
    │        │
    │        └──→ Фаза 2B (Прил. 2: Интерфейс + Финализация)
    │
    └──→ Фаза 3A (Прил. 3: Инфраструктура)
             │
             ├──→ Фаза 3B (Прил. 3: Индексация — 3 конвейера параллельно)
             │        │
             │        └──→ Фаза 3C (Прил. 3: Извлечение + Генерация)
             │                 │
             │                 └──→ Фаза 3D (Прил. 3: Оценка + Метрики)
             │                          │
             │                          └──→ Фаза 3E (Прил. 3: Интерфейс)
             │
             └──→ Фаза 3F (Прил. 3: Тестовый набор данных — можно начать после 3A)
```

**Параллелизм:** После Фазы 0 все три приложения могут разрабатываться одновременно разными агентами. Внутри Приложения 3 три конвейера индексации (документация, код, API-спецификации) также могут выполняться параллельно.

## Сводка назначения агентов

| Агент | Фазы | Объём работ |
|---|---|---|
| **Агент-0** | Фаза 0 | Общая настройка (быстро, разблокирует остальных) |
| **Агент-1** | Фаза 1A → 1B | Приложение 1 полностью |
| **Агент-2** | Фаза 2A → 2B | Приложение 2 полностью |
| **Агент-3** | Фаза 3A → 3B (параллельные подзадачи) → 3C → 3D → 3E → 3F → 3G | Приложение 3 полностью |

**Общий параллелизм:** После завершения Фазы 0 три агента работают одновременно. Агент-1 и Агент-2 закончат значительно раньше Агента-3 (Приложение 3 примерно в 3 раза больше). Освободившиеся агенты могут помочь с подзадачами Приложения 3 (например, Агент-1 берёт Фазу 3F тестового набора данных, пока Агент-3 работает над индексацией 3B).

**Критический путь:** Фаза 0 → Фаза 3A → Фаза 3B → Фаза 3C → Фаза 3D → Фаза 3E → Фаза 3G (Приложение 3 определяет общее время завершения).
